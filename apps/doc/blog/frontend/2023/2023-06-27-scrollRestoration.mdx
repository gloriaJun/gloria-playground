`window.history.scrollRestoration` 속성을 통하여 "뒤로 가기" 이벤트(history.back())가 발생하였을 때, 이전 페이지에서 사용자가 스크롤 했던 위치로 돌아갈지에 대한 여부를 설정할 수 있다.

```javascript
// 사용자가 스크롤한 페이지의 위치로 복원된다
window.history.scrollRestoration = 'auto'; // default

// 페이지의 위치는 복원되지 않으므로, 해당 위치로 수동으로 스크롤해야 한다.
window.history.scrollRestoration = 'manual';
```
