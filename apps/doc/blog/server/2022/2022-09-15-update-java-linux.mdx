설치 가능한 java 버전 확인

<!--truncate-->

```bash
yum list java*jdk-devel
```

설치하기

```bash
yum install -y java-11-openjdk-devel.x86_64
```

설치한 자바 버전으로 변경하기

```bash
/usr/sbin/alternatives --config java
```
